### Khởi chạy project :
#### Front-End : 

```
cd client
npm install

npm run dev
```

#### Back-End và kết nối database
Vào thư mục api tạo file `.env`
Thay `<username>` và `<password>` trong database
```
MONGO_URL = mongodb+srv://<username>:<password>@blog-mern.hpy2uaa.mongodb.net/blog?retryWrites=true&w=majority
```

```
cd api
npm install

npm start

```
